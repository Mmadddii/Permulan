# Permulan

**Permulan** R paketeak datuen azterketa estatistikorako oinarrizko funtzioak eskaintzen ditu:
- Atributuen **bariantza**, **entropia** eta **AUC** kalkulua  
- Atributuen **filtraketa** metrika horien arabera  
- Datu jarraituen **diskretizazioa** (*equal width* eta *equal frequency* metodoekin)  
- Atributuen **normalizazioa** eta **estandartizazioa**  
- **Korrelazio** eta **informazio mutuoko** matrizeak kalkulatzea eta bistaratzea  

## Instalazioa

```r
# Zuzenean GitHubetik
devtools::install_github("Mmadddii/Permulan")
```
